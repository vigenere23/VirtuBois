# VirtuBois Editor

![VirtuBois screenshot](https://user-images.githubusercontent.com/32545895/57198755-79986300-6f44-11e9-87ca-74f1bad1140a.png)

## Prerequisites

- Java SDK 11
- Maven

## Executing

**Compiling**

```shell
mvn clean compile
```

**Packaging**

```shell
mvn clean package
```

**Executing jar**

```shell
java -jar bin/virtubois.jar
```
